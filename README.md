# VoyageVista
Bienvenue sur me dépôt de Voyage vista de Baptise Martinat, Victor Frege, Aurelien Rubillon, Jaime miguel

## Instructions d'installation

Pour faire fonctionner ce projet en local, assurez-vous d'avoir installé Wamp

### 1. Préparation des fichiers
*   Téléchargez ou clonez ce dépôt sur votre machine.
*   Placez le dossier du projet dans le répertoire `www` de votre installation Wamp 
*   Vérifiez que le nom du dossier dans `www` est bien `voyagevista`.

### 2. Configuration de la base de données
1.  Lancez Wamp et vérifiez que les services (Apache et MySQL) sont actifs 
2.  Accédez à phpMyAdmin (http://localhost/phpmyadmin/) depuis votre navigateur.
3.  Créez une nouvelle base de données.
4.  Cliquez sur l'onglet **Importer** et sélectionnez le fichier `.sql` fourni dans ce dépôt pour charger les tables nécessaires.


### 3. Accès au site
Une fois la base de données importée, ouvrez votre navigateur et accédez à l'adresse suivante :
http://localhost/voyagevista/
